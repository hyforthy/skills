# Skills

A collection of Claude Code skills.

## Skills

- **session-viewer** — Local web UI for browsing Claude Code conversation history
- **url2markdown** — Convert any URL to clean Markdown (WeChat, Feishu/Lark, YouTube, web pages)
- **image-generator** — AI image generation expert: auto-parse subject/style/ratio/constraints, build high-quality prompts, and call image models (Seedream by default, or Gemini)

## Installation

```bash
npx skills add hyforthy/skills --skill <skill-name>
```

Replace `<skill-name>` with the desired skill (e.g. `session-viewer`, `url2markdown`, `image-generator`).
