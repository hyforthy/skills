# 完整风格词库

## 零、核心维度速查

| 维度 | 作用 | 关键决策 |
|-----|------|---------|
| 主体（Subject） | 画面核心内容 | 具象化抽象描述 |
| 风格（Style） | 美学定位 | 最多2种叠加 |
| 比例（Aspect Ratio） | 构图比例 | 从内容推断 |
| 约束（Constraints） | 正向+负向限制 | 显式写入prompt |

---

## 一、单一风格模块

### 1. 写实摄影风（Photorealistic）
```
超写实摄影风格，真实物理质感，皮肤与材质细节清晰可见，摄影棚级布光效果，85mm镜头拍摄，浅景深（depth of field），背景自然虚化（bokeh），HDR高动态范围，真实光影还原，细腻纹理，画面锐利清晰，专业商业摄影级质感
```

### 2. 电影感风格（Cinematic）
```
电影级画面风格（cinematic），宽银幕构图（cinematic framing），强戏剧性光影对比，体积光（volumetric lighting）穿透空间，环境雾气增强层次，镜头光晕（lens flare），电影调色（青橙色调 / teal & orange），浅景深，情绪张力强，电影剧照质感
```

### 3. 商业插画风（Illustration）
```
现代商业插画风格，扁平化设计（flat design），清晰几何形状构成，干净轮廓线条，高对比配色体系，视觉层级明确，构图平衡，信息表达清晰，适用于海报与社交媒体视觉设计
```

### 4. 动漫风格（Anime）
```
日系动漫风格（anime style），精致线稿（clean line art），赛璐璐上色（cel shading），高饱和色彩，柔和渐变阴影，发丝细节丰富，角色表情生动，画面具有动画感与故事性
```

### 5. 游戏概念艺术风（Concept Art）
```
游戏概念艺术风格（concept art），史诗级场景设计，世界观构建完整，结构复杂且合理，空间透视强烈（atmospheric perspective），光影营造氛围，细节密度高，具有电影级视觉冲击力
```

### 6. 赛博朋克风（Cyberpunk）
```
赛博朋克风格，未来都市夜景，霓虹灯光（neon lighting），蓝紫+洋红色调，高对比光影，潮湿反光地面，全息投影（hologram），科技感密集，反乌托邦氛围，强视觉冲击
```

### 7. 蒸汽波风格（Vaporwave）
```
蒸汽波风格（vaporwave），80年代复古未来主义，粉紫渐变天空，霓虹网格地面，VHS模拟质感，轻微故障艺术（glitch effect），梦幻怀旧氛围，数字雕塑与古典元素融合
```

### 8. 油画风格（Oil Painting）
```
古典油画风格，厚重笔触明显（visible brush strokes），丰富颜料层次，强明暗对比（chiaroscuro），构图严谨，艺术馆级质感，具有历史绘画风格特征
```

### 9. 水彩手绘风格（Watercolor）
```
水彩画风格，色彩自然晕染，边缘柔和扩散（bleeding edges），纸张纹理清晰可见，轻盈通透感强，色彩柔和，手绘艺术质感明显
```

### 10. 3D渲染风（CG / Unreal / Octane）
```
高质量3D渲染风格，PBR真实材质系统，光线追踪（ray tracing），全局光照（global illumination），反射与折射真实自然，细节精密，类似Unreal Engine / Octane Render级别
```

### 11. 极简设计风（Minimalism）
```
极简主义风格，画面高度简化，留白充足（negative space），低元素密度，几何构图清晰，单色或低饱和配色，视觉焦点极其明确，高级设计感
```

### 12. 超现实主义风（Surrealism）
```
超现实主义风格，梦境般场景结构，不符合现实物理逻辑，空间扭曲与重构，象征性表达强烈，视觉隐喻丰富，艺术表现力极强，具有哲学或心理暗示
```

### 13. 吉卜力风（Ghibli / Studio Ghibli Style）
```
吉卜力动画风格（Studio Ghibli style），宫崎骏美学，手绘动画质感，温柔自然光线，细腻草地与天空，柔和暖色调，场景充满生命力，人与自然和谐共处，梦幻奇幻氛围，细节丰富但不压迫，情感温暖治愈
```

## 三、场景应用速查

- **AI科普封面** → 电影感 + 赛博朋克
- **知识类封面** → 插画 + 极简
- **人物表达** → 电影感 + 写实摄影
- **情绪表达** → 超现实 + 电影感
- **产品展示** → 3D渲染 + 科技风
- **治愈内容** → 吉卜力风 / 水彩 + 插画
- **奇幻动画场景** → 吉卜力风 + 电影感
- **游戏/幻想** → 概念艺术 + 电影感
- **复古潮流** → 蒸汽波 + 插画

---

## 四、风格使用规则

1. **单用**：一个风格模块可直接完整使用
2. **组合**：最多 1主风格 + 1辅助风格，超过2种风格会产生冲突
3. **优先级**：情绪表达 > 风格词 > 细节词 > 质量词
